# Full Stack Blog App

A full stack blog application built with React, Node.js, Express, and MongoDB.

## Features
- Create blog posts
- Edit blog posts
- Delete blog posts
- Data saved to MongoDB database
- Clean and responsive UI

## Tech Stack

**Frontend:**
- React
- Axios
- CSS3

**Backend:**
- Node.js
- Express
- MongoDB + Mongoose
- CORS
- dotenv
